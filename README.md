# CommerceHub — E-Commerce Admin Dashboard

![React](https://img.shields.io/badge/React-Frontend-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Dashboard](https://img.shields.io/badge/Admin-Dashboard-green)
![Status](https://img.shields.io/badge/Status-Active-success)

CommerceHub is a modern and responsive **E-Commerce Admin Dashboard** designed to simplify store management through an intuitive interface. The dashboard enables administrators to manage products, orders, customers, inventory, analytics, and application settings from a centralized platform.

 **Live Demo:** [CommerceHub](https://commercehub-admin-omega.vercel.app/) </br>
 **GitHub Repository:** https://github.com/RadhikaKapoor383/CommerceHub-Admin

---

## Project Overview

CommerceHub provides administrators with a complete dashboard experience for monitoring and managing e-commerce operations. The application offers a clean user interface, responsive design, and data-driven insights to improve business decision-making.

The project focuses on creating reusable UI components, efficient data management, and an engaging dashboard experience.

## Installation

### Clone the repository

```bash
git clone https://github.com/RadhikaKapoor383/CommerceHub-Admin.git
```

### Navigate to the project folder

```bash
cd CommerceHub-Admin
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

---

## Run Locally

```bash
npm install
npm start
```

The application will run at:

```text
http://localhost:3000
```
